## 👤 User Journey

1. Student opens MindBridge
2. Creates an account / Logs in
3. Completes a quick onboarding
4. Selects today's mood
5. Gets personalized wellness suggestions
6. Can try a breathing exercise or journal
7. If more support is needed:
   - Connect with a Senior Buddy
   - Connect with a Mentor
   - Book a Professional Counselor
8. Track progress on the dashboard
